# **App Name**: ArtNft

## Core Features:

- Content Creation: Simple text editor for content creation.
- Content Display: Displays existing textual content.
- Admin Management: Allows adding, deleting and editing existing text.
- AI Text Improvement: AI-powered tool suggests improvements to text based on the text tone.

## Style Guidelines:

- Primary color: Soft sky blue (#87CEEB) to evoke calmness and readability.
- Background color: Light, desaturated blue (#E6F3FF) for a clean interface.
- Accent color: Warm coral (#FF7F50) for interactive elements.
- Headline font: 'Literata', serif, for a literary feel.
- Body font: 'Inter', sans-serif, providing clarity and readability.
- Simple, line-based icons for a modern aesthetic.
- Clean, minimalistic layout for easy content focus.
- Subtle transitions on text edits and AI suggestions.
